Virtual Reality as a Tool for Improving Study Performance

(Master's Thesis on Cognitive Science)



Project Goal:

  • Goal: To investigate whether there is an influence of recognition of virtual environments on improving learning performance.

  • Implementation Duration: 4 Months
⭐ Description

For this research, I have developed a minimum viable product, a VR prototype, that consists of multiple Virtual Environments. This prototype includes the necessary features to study both online and offline. However, in terms of research, multiplayer wasn't implemented.

Using this application, students can watch pre-recorded lectures in a virtual classroom simulation.

⭐ What I did as a Level Designer
  • APPLICATION DEVELOPMENT AND OPTIMIZATION:
    Implemented a VR classroom prototype with two virtual classes;

  • ENVIRONMENTAL DESIGN:
    Designed and crafted immersive virtual environments using various techniques;

  • IN-GAME MECHANICS:
    Implemented custom C# scripts to extend the capabilities of XR Interaction Toolkit;

  • IMPLEMENTED FEATURES:
    Successfully implemented features such as video player, wrist UI menu, seat selection, sound volume control, control timeline, locomotion system, camera reset button, spatial audio, multiple NPCs, lighting, and controller bindings;

  • OTHER IMPLEMENTATIONS:
    Selected Oculus Quest 2 as the target device for implementation and testing; Utilized GitHub for version control and collaboration during development; Utilized Sketchfab, Mixamo, and Ready Player Me plugins for realistic characters and animations.
