Tags

Obstacles

  • Type 1 - Simple Challenges: a) Obstacles which our player must climb using jumps or pass under using bend button to complete level (e.g. Big tree, upper bounds) b) Steep cliff / pit which player must avoid or overcome c) lava, lightning, acid rains, darkness, etc.
  • Type 2 - Enemies: a) lava rocks b) ghosts c) skeletons d) demons e) some other creatures
  • Type 3 - Mini tasks/games: a) find a key b) enter password or keyword c) pay an entrance fee etc ...
  • Type 4 - Villain/Boss: mad Magician etc...
  • Type 5 - Collected points: points give access to further lvls (collected with: coins, killing mobs, upgrades)
  • Type 6 - Limited number of slots in inventory (e.g. a player needs to drop smth in order to pick up smth else)

Loot

  • Type 1 - Coins (stored in: Coin Counter)
  • Type 2 - New weapons, clothes
  • Type 3 - Not-playable items (e.g. keys, magic spells, papers)

Levels

  • Level 1 - Forest/Village
  • Level 2 - Volcano
  • Level 3 - Graveyard
  • Level 4 - Crypt
  • Level 5 - Final Level
